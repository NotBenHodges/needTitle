Cairo's Techniques

1. Decomposing a Time Series- Many graphs use straight lines to display general trends but that
does not give an accurate depiction of the trend. Instead, we can implement a line that
follows the trend nonlinearly by basing it on a moving average. We can split the time series into
chunks and calculate the average of each chunk to give us a more accurate depiction of the trend.

2. Visualizing Indexes- Time series data can also be represented using zero-based indexes.
These indexes can be calculating by using the formula for percent change:

% Change = ((Each Score - Index Origin) / Index Origin) x 100

3. Logarithmic Transformation- If the current chart does not give an overall idea of the change rate,
then a log Transformation can be helpful. A base is decided that determines the increase of each unit.
For example, a log 2 scale means that each incremental increase in the graph is not 1 additional increment, but
twice the size of the unit before it.
